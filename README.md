# QLDT PTIT Calendar Tool

## 🔰Giới thiệu

Bạn là sinh viên PTIT và bạn muốn quản lý thời khóa biểu của mình một cách hiệu quả, nhưng bạn muốn đồng bộ thời khóa biểu của mình với Google Calendar để tiện theo dõi. Vậy thì bạn đã tìm đúng chỗ rồi.

## 🚀 Tính năng

- Đồng bộ thời khóa biểu với Google Calendar
- Giao diện calendar nhiều màu sắc
